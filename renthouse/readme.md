# 租屋 vs 買房真實投報率計算機 (Rent vs Buy ROI Calculator)

## 專案簡介
本專案為一頁式 Web 應用程式，旨在打破「租屋是幫房東繳房貸」的傳統迷思。透過量化「真實房貸利率」與「資金機會成本（如 0050 ETF 預期報酬）」，精算租屋與買房的真實資產變化，輔助使用者做出理性的財務決策。

## 核心功能
* **雙模式試算：**
  * **模式一（精確計算）：** 輸入已知總價與租金，直接進行現金流對決。
  * **模式二（區域估算）：** 內建全台 22 縣市、近 80 個重點行政區的 2025-2026 最新實價登錄均價，輸入坪數即可自動估算總價並評估。
* **真實機會成本模型：** 捨棄傳統粗略的「房價租金比」，將「8 成房貸純利息」與「2 成頭期款投入 0050 的 5% 預期獲利」合併計算為買房真實資金成本，與年租金進行絕對值對決。
* **五級投報率判定：** 根據現金流落差，提供從「買爆它（完勝 0050）」到「無腦租屋（資產黑洞）」的明確決策建議與金額對比。

## 技術架構
* **前端介面：** HTML5 + Tailwind CSS (經由 CDN 載入，無須額外編譯)。
* **邏輯運算：** Vanilla JavaScript (純原生 JS，無依賴第三方框架)。
* **SEO 與社群分享：** 具備完整 Meta Tags、Open Graph (OG) 與 Twitter Cards 標籤。

## 使用方式
1. 將專案中的 `index.html` 下載至本地端。
2. 使用任何現代瀏覽器（Chrome, Edge, Safari）直接開啟該檔案即可運行。
3. 若需修改各地區預估單價，請直接使用文字編輯器開啟 `index.html`，搜尋 `<select id="region">` 區塊修改 `value` 數值（單位：萬/坪）。
