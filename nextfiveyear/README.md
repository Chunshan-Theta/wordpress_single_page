# 人生模擬器 - 預見你的未來可能性

這是一個互動式的網頁模擬器，幫助用戶根據當前的生活條件模擬未來 5 年的收入成長、生活品質指標以及個人化建議。

## 功能特色

- **年齡滑桿**：選擇 18–70 歲，模擬不同年齡的未來可能性。
- **職業選擇**：工程師、設計師、教師、業務、學生、退休等。
- **月收入輸入**：根據當前收入進行模擬。
- **家庭狀況**：單身、已婚、有小孩。
- **所在城市**：台北、新北、台中、高雄等。
- **模擬情境**：
  - 只有原工作（灰色卡片）
  - 原工作 + 被動收入一萬（藍色卡片）
  - 被動收入五萬（金色卡片）
- **結果展示**：
  - 收入時間軸（3 個月、1 年、3 年、5 年）
  - 收入成長圖表
  - 生活品質指標（工作時間彈性、健康指數、家庭時間等）
  - 個人化建議文字
- **特色功能**：
  - 週休三日模擬器
  - 反社畜指數計算器
  - 分享結果到社群媒體
- **設計風格**：
  - 現代科技感設計
  - 藍色和金色主色調
  - 圓潤的按鈕和卡片設計
  - 動畫效果和漸變色
  - 響應式設計，手機友好

## 使用方式

1. **嵌入 WordPress**：
   - 將 `life-simulator.html` 的內容複製到 WordPress 的「自訂 HTML」區塊中。
   - 如果 WordPress 過濾 `<script>` 標籤，請安裝 **WPCode** 或 **Raw HTML** 外掛。

2. **本地測試**：
   - 使用任意瀏覽器打開 `life-simulator.html`。

3. **自訂樣式**：
   - 修改 CSS 樣式表中的 `:root` 變數以調整配色。
