# 修仙模擬器 — 問道蒼穹

互動式文字角色扮演遊戲。選擇天賦、承受宿命烙印，踏上修仙之路，看看你能突破哪一境界。

## 玩法說明

1. **選擇天賦**：從隨機抽出的 4 張天賦卡中選擇 2 張
2. **接受烙印**：系統隨機指派一個宿命烙印（負面特性），無法迴避
3. **踏上道途**：點擊「渡過一年」，每次隨機經過 2～4 年並觸發一個隨機事件
4. **觀察數值**：五維數值任一歸零即隕落，越晚死越厲害
5. **查閱墓誌銘**：死亡後顯示完整生涯總結與事件紀錄
